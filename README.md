# How to Run
1. In the terminal window, run
    `yarn dev`
OR
1. In one terminal window, run
    `ng serve`
2. In a separate terminal, run
    `json-server --watch db.json`
2. Go to [http://localhost:4200/](http://localhost:4200/)


# Features
- Home page
    - World information
- Timeline
- World
- Characters
- Stories
